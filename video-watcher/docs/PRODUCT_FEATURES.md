# Video Watcher - Product Features and Roadmap

## Current Features

### Core Functionality
- Real-time video monitoring and detection
- Integration with Google's Gemini AI
- Automated video analysis and labeling
- Web-based dataset exploration interface
- OBS Studio integration

### Data Management
- JSON-based dataset storage
- Structured metadata generation
- Basic search and filtering capabilities
- Dataset export functionality
